# 🔐 CIPH3RX7 – Real-Time Hardware Cipher Engine

A real-time hardware encryption platform that combines **Cybersecurity**, **Embedded Systems**, and **Digital Electronics** to demonstrate secure communication using physical XOR logic instead of software-only encryption.

The project showcases how hardware circuits can actively participate in data encryption while providing live monitoring and traffic visualization.

---

# 🚀 Project Overview

CIPH3RX7 is designed to perform real-time encryption by integrating:

✅ Arduino UNO

✅ Hardware XOR Encryption Circuit

✅ Ring Oscillator

✅ Python Communication

✅ Real-Time Dashboard

✅ Packet Counter

✅ Live Ciphertext Visualization

# 🔐 Encryption Principle

Encryption uses hardware XOR logic:

Ciphertext = Plaintext XOR Key

The same XOR operation is used again for decryption.
